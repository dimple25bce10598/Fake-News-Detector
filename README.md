Fake News Detector using Machine Learning

Overview:
The Fake News Detector is a project that uses machine learning to figure out if a news article or message is real or fake. It looks at the text. Decides what it is. This project helps stop news from spreading and makes people more aware of what is real and what is not.
________________________________________

Features:
* The Fake News Detector can tell if news is real or fake
* It uses machine learning to make this decision
* The Fake News Detector is simple to use and easy to understand
* You do not need any data to use the Fake News Detector
* It works fast. Does not take up a lot of space
________________________________________

Technologies Used:
* Python is used to make the Fake News Detector
* Pandas is used to work with data
* Scikit-learn is used for machine learning
* Natural Language Processing is used to clean and prepare the text data
________________________________________

Installation:
1. Get the code from the internet. Download it
2. Install the libraries that the Fake News Detector needs:
      pip install pandas scikit-learn
3. Run the Python file:

      python fake_news_detector.py
________________________________________

How the Fake News Detector Works?
1. The Fake News Detector makes a set of real and fake news
2. The text is cleaned up
       * The text is made lowercase
       * Punctuation and special characters are removed
 3. The text is turned into numbers
       * The Fake News Detector uses TF-IDF to do this
4. The model is trained
       * The Fake News Detector uses a kind of classifier
  5. The Fake News Detector makes a prediction
        * You enter some news text
        * The Fake News Detector says if it is real or fake

Using the Fake News Detector:
After you start the program enter some news text:
